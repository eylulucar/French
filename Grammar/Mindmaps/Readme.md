## Le Passé Composé ⏳

### Uses

- être

	- Je suis ✔️
	- Tu **es** ❌ (correct form is "es")
	- Il est ✔️
	- Nous sommes ✔️
	- Vous êtes ✔️
	- Ils sont ✔️

- avoir

	- J'ai ✔️
	- Tu **as** ❌ (correct form is "as")
	- Il **a** ❌ (correct form is "a")
	- Nous avons ✔️
	- Vous avez ✔️
	- Ils ont ✔️
